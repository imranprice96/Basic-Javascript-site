Javascript Part 2

Loops

Like many other languages, Javascript can use loops to execute codes of block.

For loop

Loops through a block of code a number of times
for(i = 0; i < 10; i++){
number=i;
}

While loop

Loops through a block of code while the condition is true
while(color != "red"){
doSomething();
}

Conditions

Conditional statements are for when you want to perform an actions based on a decision.

If statement

The code is executed "if" the statement is true

Else statement

The code is executed if the condition is false.

Functions

Javascript functions are blocks of code made for performing specific tasks.
The code is run when the function is called.
Functions can require parameters such as arguments and values.
Functions can return values and can call functions within themselves.
