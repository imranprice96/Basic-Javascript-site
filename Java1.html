Javascript Part 1

Data Types

The three main data types in Javascript are:
Numbers
Strings
Boolean

Variables

Variables in Javascript are dynamically typed.
Javascript variables can hold any data type Variables can be declared with the "var" or "let" keywords.

Examples:

let car = toyota;
var animal;
animal = cat;
var person, name, age;

Strings

Because Javascript treats everything as objects, primitive values can have methods and properties

Examples:

var str = "abcdefghijklmnopqrstuvwxyz";
var num = str.length;
